# Python GUI Calculator

A modern calculator app written in Python and built with Tkinter that runs on macOS, Windows, and Linux.

## Features
The app provides standard calculator functionality:
- Unary operations (+/-, %)
- Binary operations (+, -, *, /)
- Keyboard inputs
